# 100Hires Portfolio Project

## Project Objective

This repository documents the setup process completed as part of my application for the Junior Growth Marketing Specialist position through 100Hires.

The objective of this project was to follow the required technical setup, document the tools installed, track the steps completed, and demonstrate how I approached changes in tool interfaces with a problem-solving mindset.

As someone who regularly uses AI tools for research, learning, productivity, content development, analysis, and problem-solving, I approached this task as an opportunity to show adaptability, initiative, and comfort working in AI-assisted environments. My familiarity with Codex comes from my active use of ChatGPT as part of my personal productivity stack, including a paid plan that I use to strengthen my workflow, accelerate learning, and support structured execution.

## Challenges Encountered

While following the setup instructions, I noticed that the current Cursor interface appeared to differ from the one described in the original guidance.

The marketplace available in my installation focused on Skills and MCPs, and I was not able to locate the specific Claude Code and Codex extensions mentioned in the email under those exact names.

Rather than treating this as a blocker, I used it as an opportunity to investigate the updated interface, explore the available tooling, and determine the best way to continue completing the objective.

## Investigation and Resolution

To move forward, I reviewed the available Cursor marketplace experience, explored the AI-related options presented in the current interface, and compared them against the expected setup flow.

Because AI tools are already part of my daily workflow, I leveraged my experience with ChatGPT and Codex to organize the project documentation, clarify the structure of the README, and ensure the final output communicated the process professionally. This reflects how I use AI in a practical way: not as a replacement for decision-making, but as a tool to improve speed, clarity, and execution quality.

This approach allowed me to keep momentum, document the discrepancy clearly, and still complete the core purpose of the task: demonstrating technical curiosity, AI fluency, and the ability to adapt when tools or interfaces change.

## Outcome
